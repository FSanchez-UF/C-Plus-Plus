Detailed Description

Finding level count of tree using a queue.

Create a queue storing the root of the tree. Enters while loop that loops until the queue is empty. Then enters for loop that loops current size of queue which holds the nodes in the same level. Checks if current node has left or right child and if child exists, adds the child to the queue. Once for loop is over, it increases level count since the next iteration of the queue will be the next level in the tree. When queue is empty, the tree has been fully traversed and the level count is displayed.
